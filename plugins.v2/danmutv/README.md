# 弹幕刮削(影视版)

此插件基于 [HankunYu/MoviePilot-Plugins](https://github.com/HankunYu/MoviePilot-Plugins) 原弹幕刮削插件修改，通过 [danmu-api](https://github.com/huangxd-/danmu_api) 从影视源获取弹幕数据，将弹幕转换为 ASS 格式字幕文件，用于在不支持弹幕播放的设备上模拟弹幕效果。

### 功能特性

- 支持电视剧、电影、动漫等多种媒体类型的弹幕获取
- 通过 TMDB ID 自动匹配影视弹幕，支持手动匹配和集数偏移
- 多层弹幕效果：可选 2 层/3 层，不同速度、透明度和字体大小
- 弹幕密度控制和同屏上限，可调节弹幕宽度扩展
- 彩色弹幕优先显示在顶层，支持随机顶/底部弹幕
- 智能缓存策略：本地已有弹幕文件跳过 API 请求，已有提取字幕跳过重新提取
- 批量刮削限流优化：串行执行 + 指数退避重试机制
- 支持内嵌字幕提取与合并，生成 `withDanmu.ass` 合并字幕文件

### 手动匹配与搜索

- 前端支持搜索弹幕资源并手动匹配，可选"目录"或"单文件"范围；单文件仅作用于该文件
- 手动匹配数据保存在目录下 `.dandan.anime.json`；单文件匹配存储在插件缓存
- 手动匹配支持"集数偏移"：本地集数 + 偏移 = 弹幕集数
