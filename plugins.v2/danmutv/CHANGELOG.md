## [1.5.0] - 2026-07-23
### Added
- 新增清理字幕功能按钮，可一键清理刮削路径下所有弹幕和合并字幕文件，支持强制重新获取最新弹幕
- 新增本地弹幕文件缓存复用：已有 `.danmu.chs.ass` 时跳过API请求，直接复用本地文件进行合并

### Changed
- 优化内嵌字幕提取逻辑：已有本地提取的 `.chs.ass` 字幕文件时跳过重新提取
- 优化批量刮削缓存策略，大幅减少重复API请求和字幕提取操作

### Fixed
- 修复合并字幕时只提取一条样式导致多层弹幕效果丢失的问题
- 修复批量刮削限流问题：串行执行+指数退避重试+全局冷却机制

## [1.4.0] - 2026-07-22
### Added
- 新增弹幕层数可选设置（2层/3层），默认2层，解决3层弹幕过厚过立体问题
- 2层模式分配比例：顶层25% / 中层75%
- 新增弹幕宽度扩展设置项（1.0x-2.0x），解决超宽屏/手机屏左右空白问题
- 新增前端弹幕密度设置选项，支持随机保留指定比例弹幕

### Changed
- 彩色弹幕优先分配到顶层显示，突出彩色弹幕视觉效果
- 多层弹幕开关描述优化，层数选择采用按钮组形式

### Fixed
- 修复 `width_scale` 参数未传递到弹幕生成函数导致宽度扩展不生效的问题
- 修复 `_save_config` 中 `update_config` 调用缺少 `density`、`width_scale` 字段的问题
- 修复前端设置页面密度选项未正确显示的问题（重新编译dist）

## [1.3.0] - 2026-07-22
### Added
- 添加1/3屏弹幕显示区域选项，介于半屏和1/4屏之间
- 添加随机顶/底部弹幕功能，可配置顶部和底部比例（最大10%）
- 添加弹幕计数日志，统计API返回数、写入数及各丢弃原因数量

### Changed
- 重写碰撞检测算法，从时间检查改为间隙检测（buffer=15px），弹幕丢弃率从86%大幅降低
- 多层弹幕每层独立轨道字典和轨道数，轨道总容量提升3倍
- 统一单层/多层模式碰撞检测为 find_non_overlapping_track_v2，新增追赶检测

### Fixed
- 修复 `\move` 目标与 `text_width` 计算不一致导致速度计算错误的bug
- 修复多层弹幕三层共享 `scrolling_tracks` 导致轨道容量浪费的问题
- 修复多层弹幕 `max_tracks` 基于 `fontsize` 计算但顶层用 `fontsize*1.2` 导致轨道超出屏幕的问题

## [1.2.0] - 2026-07-22
### Added
- 添加多层弹幕功能，弹幕分为三层显示
- 添加启用多层弹幕配置开关
- 添加改进的碰撞检测算法，支持速度感知的轨道分配

### Changed
- 修改 `write_ass_head` 方法支持生成3个层次样式定义
- 修改 `convert_comments_to_ass` 方法实现多层弹幕生成逻辑
- 修改 `danmu_generator` 函数支持传递多层弹幕开关参数
- 更新前端配置页面添加多层弹幕开关控件
- 调整多层弹幕分层比例：顶层25%→20%，中层50%→55%，底层25%不变
- 调整顶层字体倍率1.1x→1.2x，增强主角弹幕视觉突出度
- 调整中层透明度倍率0.9→0.8，强化顶层与中层的对比度
- 调整默认透明度0.8→0.9，提升整体可读性
- 统一_fontsize类属性默认值48→50，与配置项默认值一致

### Fixed
- 修复缺少 `import random` 导致多层弹幕处理全部失败的错误
- 修复 `_save_config` 方法中残留 `onlyFromBili` 属性引用导致保存配置失败

### Removed
- 无
